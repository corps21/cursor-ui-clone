# Cursor UI Clone

This is a recreation of the Cursor AI editor's landing page using HTML and CSS.

## Recreated Sections

The following sections from the original Cursor landing page have been recreated:

- **Navigation Header**: Logo, navigation links (Features, Enterprises, Pricing, Resources), and sign-in/download buttons
- **CTA Section**: Main headline and download button for Linux
- **Trusted By Section**: Logos of companies that trust Cursor (Stripe, OpenAI, Linear, Datadog, Nvidia, Figma, Ramp, Adobe)
- **Feature Section**: Three feature cards showcasing:
  - Agent turns ideas into code
  - Magically accurate autocomplete
  - Everywhere software gets built
- **Testimonial Section**: Grid of testimonials from industry leaders including Diana Hu, shadcn, Andrej Karpathy, Patrick Collison, ThePrimeagen, and Greg Brockman
- **Product Overview Section**: Three cards highlighting:
  - Access the best models
  - Complete codebase understanding
  - Develop enduring software
- **Changelog Section**: Recent updates with version badges and dates
- **Join Us Section**: Team-focused section with call-to-action
- **Recent Highlights Section**: Blog-style posts about recent developments
- **Try Now Section**: Final call-to-action for downloading Cursor
- **Footer**: Comprehensive footer with product links, resources, company info, legal links, and social media connections

## Fonts Used

- **Primary Font**: CursorGothic (custom font)
  - Regular (400 weight)
  - Italic (400 weight, italic style)
  - Bold (700 weight)
  - Bold Italic (700 weight, italic style)
- **Fallback Fonts**: Segoe UI, Tahoma, Geneva, Verdana, sans-serif

Font files are located in the `fonts/` directory:
- CursorGothic-Regular.woff2
- CursorGothic-Italic.woff2
- CursorGothic-Bold.woff2
- CursorGothic-BoldItalic.woff2

## Colors Used

The design uses a dark theme with the following color palette:

### Primary Colors
- **Primary Background**: `#14120b` (very dark brown/black)
- **Secondary Background**: `#1b1913` (slightly lighter dark brown)
- **Text Color**: `#edecec` (off-white/light gray)

### Accent Colors
- **Link Color**: `#f54e00` (orange)
- **Link Hover**: `color-mix(in oklab, #f54e00 75%, transparent)` (semi-transparent orange)

### UI Element Colors
- **Muted Text**: `color-mix(in oklab, #edecec 60%, transparent)` (60% opacity of text color)
- **Button Secondary Border**: `color-mix(in oklab, #edecec 20%, transparent)` (20% opacity of text color)
- **Card Border**: `color-mix(in oklab, #edecec 2.5%, transparent)` (2.5% opacity of text color)
- **Card Hover**: `#1d1b15` (dark brown hover state)
- **Highlight Section Card**: `#201e18` (medium dark brown)
- **Highlight Section Card Hover**: `#26241e` (lighter medium dark brown)
